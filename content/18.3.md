En esta práctica final, vamos a aplicar la metodología de Spec-Driven Development (SDD). En lugar de abrir un archivo **`.py`** y empezar a codificar a ciegas, vamos a comportarnos como verdaderos arquitectos: primero definimos las reglas y la intención, y luego dejamos que el Agente (nuestro editor nativo de IA) genere la implementación.
<br><br>
Como vimos en la tarjeta anterior, este flujo estricto elimina el "impuesto de ambigüedad". Para hacerlo ágil, vamos a consolidar las fases de *Constitution*, *Specification*, *Plan* y *Tasks* en un único documento maestro. Este archivo Markdown es nuestro contrato y la única fuente de verdad a partir de la cual se derivará el código.
<br><br>
El flujo de trabajo es así de simple: escribís el documento **`spec.md`**, abrís tu AI Code Editor (como Cursor o Windsurf), le pasás este archivo al agente en el chat lateral y le das la orden: *"Implementá app.py basándote estrictamente en spec.md"*.
<br><br>
Aca tenes el SDD para pasarle a tu ia y comprobar por vos mismo su funcionamiento:
<br><br>
```python

# Archivo: spec.md (La fuente de verdad escrita por el Arquitecto)

## 1. Constitution (Reglas de Arquitectura)
- Lenguaje: Python puro.
- Dependencias permitidas: `google-genai`, `python-dotenv`.
- Seguridad: Estrictamente prohibido hardcodear API Keys. Cargar credenciales desde `.env`.

## 2. Specification (La Intención)
- Objetivo: Crear un script de consola interactivo que simule un Agente de Soporte usando la nueva API de Interactions de Gemini.
- Comportamiento: El usuario ingresa un problema. El agente debe responder con un tono ultra-profesional. Si el usuario escribe "salir", el script termina de forma segura.

## 3. Plan & Tasks (Desglose Atómico)
- [ ] Tarea 1: Configurar la carga de variables de entorno y validar seguridad.
- [ ] Tarea 2: Inicializar el cliente de Gemini de forma local (`genai.Client()`).
- [ ] Tarea 3: Crear el bucle `while True` para capturar el input del usuario en consola.
- [ ] Tarea 4: Ejecutar la inferencia inyectando un System Prompt para forzar el tono, e imprimir la propiedad `output_text` de la respuesta.

```
<br><br>
Al leer este documento, tu agente de IA no tiene que adivinar nada ni tomar decisiones creativas peligrosas. Sabe exactamente qué dependencias instalar, qué formato usar y qué tareas completar.
