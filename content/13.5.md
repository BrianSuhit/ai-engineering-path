A nivel de arquitectura de sistemas, cuando queremos que nuestro modelo analice una imagen local, no le enviamos la ruta del archivo. Lo que hacemos es leer los *bytes* de la imagen y codificarlos en **Base64**. Esto convierte la imagen cruda en una cadena de texto inmensa que podemos inyectar de forma segura en nuestra carga útil (*payload*) HTTP hacia la API.
<br><br>
Además, en un entorno de producción, rara vez queremos que el modelo nos devuelva un párrafo libre describiendo la foto. Si estamos construyendo una aplicación que etiqueta imágenes automáticamente, necesitamos datos estructurados. Para resolver esto, combinamos la API de *Interactions* con **Pydantic** para definir un esquema estricto. El servidor de Google intercepta este esquema y restringe el motor de generación (usando *Structured Outputs*) para garantizar que el modelo solo pueda devolver un objeto JSON válido que coincida exactamente con nuestras variables.
<br><br>
Acá tenés el flujo completo y profesional para analizar una imagen de tu computadora:
<br><br>

```python

# Paquetes a instalar en tu entorno virtual:
# uv add google-genai pydantic python-dotenv

import base64
import json
from google import genai
from pydantic import BaseModel, Field
from dotenv import load_dotenv

# Cargamos las variables de entorno
load_dotenv()
client = genai.Client()

# 1. Definimos el esquema estricto de salida usando Pydantic
class ImageAnalysis(BaseModel):
    description: str = Field(description="Una breve descripción general de la imagen.")
    dominant_colors: list[str] = Field(description="Lista de los colores predominantes.")
    objects_detected: list[str] = Field(description="Lista de objetos reconocibles en la escena.")

# 2. Cargamos la imagen local y la codificamos en Base64
# (Asegurate de tener una imagen llamada 'sample.jpg' en la carpeta 'images')
try:
    with open("images/sample.jpg", "rb") as f:
        image_bytes = f.read()
        image_b64 = base64.b64encode(image_bytes).decode("utf-8")
except FileNotFoundError:
    print("🚨 Error: No se encontró el archivo 'images/sample.jpg'. Asegurate que la imagen exista en esa ruta.")
    exit()

print("📸 Analizando imagen...")

# 3. Hacemos la llamada a la API de Interactions con la imagen y el esquema
interaction = client.interactions.create(
    model="gemini-3.5-flash",
    input=[
        {"type": "text", "text": "Analizá esta imagen en detalle y extraé la información solicitada."},
        {
            "type": "image",
            "data": image_b64,
            "mime_type": "image/jpeg"
        }
    ],
    response_format={
        "type": "text",
        "mime_type": "application/json",
        "schema": ImageAnalysis.model_json_schema()
    }
)

# 4. Parseamos el texto crudo del modelo directamente a nuestro objeto Pydantic
analysis_result = ImageAnalysis.model_validate_json(interaction.output_text)

# 5. Mostramos el resultado estructurado
print("\n🤖 Resultado Estructurado:")
print(json.dumps(analysis_result.model_dump(), indent=2, ensure_ascii=False))

```
