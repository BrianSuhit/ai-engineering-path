Llegó el momento de ensamblar todas las piezas del motor. Un sistema RAG funcional requiere orquestar la ingesta de datos, la recuperación matemática y la generación del modelo de lenguaje en un solo flujo continuo.
<br><br>
Para este ejemplo, vamos a construir un asistente universitario completo. El flujo de nuestro código hará exactamente lo siguiente:
<br><br>
1. **Ingestion:** Guardará los fragmentos de texto (simulando un PDF de la facultad, usaremos el mismo ejemplo de las materias de Brian) en ChromaDB.
2. **Retrieval:** Tomará tu pregunta en lenguaje natural, buscará en la base de datos vectorial y extraerá solo las materias relevantes.
3. **Augmentation:** Tomará esos fragmentos recuperados y los inyectará en un *Prompt Template* estructurado.
4. **Generation:** Enviará ese *prompt* final a Gemini (usando su moderna **`Interactions API`**) para que redacte la respuesta final basándose estrictamente en tus datos.
<br><br>
Acá tenés el código de la arquitectura completa funcionando de punta a punta:
<br><br>

```python
# Para instalar los paquetes necesarios, ejecuta:
# uv add chromadb python-dotenv google-genai
import chromadb
from dotenv import load_dotenv
from google import genai

# 1. Configuración del cliente de Gemini (asume que GEMINI_API_KEY está en tu .env)
load_dotenv()
client = genai.Client()

# 2. Inicializar ChromaDB (Nuestra memoria semántica)
chroma_client = chromadb.Client()
collection = chroma_client.create_collection(name="brian_university_syllabus")

# 3. Ingesta de datos (Simulando fragmentos extraídos del PDF de tu facultad)
pdf_chunks = [
    "Brian cursa Programación en el 2do cuatrimestre.",
    "Brian cursa Web 1 en el 2do cuatrimestre.",
    "Brian cursa Inglés 2 en el 2do cuatrimestre.",
    "Brian cursa Teoría de la información en las organizaciones en el 2do cuatrimestre."
]

collection.add(
    documents=pdf_chunks,
    ids=["chunk_1", "chunk_2", "chunk_3", "chunk_4"]
)

# 4. La consulta del usuario
user_query = "¿Qué materias tengo en el segundo cuatrimestre que impliquen escribir código?"

# 5. FASE DE RECUPERACIÓN (Obteniendo los 2 fragmentos más relevantes matemáticamente)
results = collection.query(
    query_texts=[user_query],
    n_results=2
)
# Unimos los fragmentos recuperados en una sola cadena
retrieved_context = " ".join(results['documents'][0])

# 6. FASE DE GENERACIÓN AUMENTADA (Ensamblando el prompt)
final_prompt = f"""
You are a helpful university assistant. Answer the student's question using ONLY the provided information below.
Do not hallucinate or invent subjects.

Retrieved Information:
{retrieved_context}

Student Question: {user_query}
"""

# 7. Llamada al LLM usando la API de Gemini
interaction = client.interactions.create(
    model="gemini-3.5-flash",
    input=final_prompt
)

# 8. Imprimir la respuesta final fundamentada
print("=== RAG SYSTEM RESPONSE ===")
print(interaction.output_text)

```
